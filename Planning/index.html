Tour at Duke

Partners: Jeff Kim (ek111), Amy Kim (yk154)

Features:

  1. List of interesting places at Duke
    • Users can filter the list with multiple categories
      1. Outdoor/indoor
      2. Location (East/Central/West)
      3. Educational/Entertainment
      4. Food
    • Automatically generate a trip with selected places
      1. Show the places on a map
      2. Show the total time it takes for the visit
  2. Sign in
    • Guest
      1. Able to browse through places at Duke to visit
      2. Able to browse through the forum
    • User
      1. Able to browse through places at Duke to visit
      2. Able to put it in their cart.
      3. Able to automatically generate a trip with the places in their cart
      4. Add new places to visit in forum so the admin can look and add if valid
    • Administrator
      1. All the features a user has
      2. Able to modify the list of places

Technology:

  1. Vue
    • Create a reactive website that could do the features stated above
  2. Firebase
    • Create new Users
    • Save user’s preference and carts in a real-time database
  3. API
    • Google maps API
      1. Displays user’s chosen places on a map
      2. Calculates the shortest trip plan that visits the chosen places
